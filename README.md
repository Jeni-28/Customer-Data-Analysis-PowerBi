# Customer Data Analysis (Power BI)

## 📊 Project Overview

This project analyzes customer shopping behavior across multiple shopping malls (2021–2023). The goal is to identify patterns in customer demographics, product categories, and payment methods to support data-driven business decisions.

An interactive Power BI dashboard was developed to visualize key insights.

## 🛠 Tools & Technologies

* Power BI
* Power Query
* DAX

## 📂 Data Source

Data was accessed through a live database connection using Power BI.
Due to access restrictions, the dataset is not publicly available.

## 📊 Key Metrics

* Total Sales: 251M+
* Total Transactions: 99K+
* Total Products Sold: 299K+

## 🔍 Key Insights

* Female customers contribute the highest revenue and transactions
* Age group 56+ shows the highest purchasing power
* Clothing is the top-performing category
* Cash is the most preferred payment method

## 📌 Features

* Interactive dashboard with slicers (Age, Gender, Category)
* KPI cards for business performance
* Multi-page analysis (Gender, Age, Category, Payment)

## 📁 Files Included

* Power BI Dashboard (.pbix)
* Project Report
* Presentation (PPT)
* Dashboard Screenshots

## 🚀 Outcome

This project demonstrates how data visualization can transform raw data into actionable business insights.
